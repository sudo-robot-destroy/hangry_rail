A collection of 3D printed mounts and hangers for garage tools designed to mount to a horizontal 1x4 or 2x4 board.

## Start here

- Browse model organization: [STL Library](stl/)
- View repository: [GitHub Repo](https://github.com/sudo-robot-destroy/hangry_rail)

## Project goals

- Keep tool storage modular and easy to reconfigure
- Share printable files and revision notes
