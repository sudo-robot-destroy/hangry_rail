# STL Library

This folder stores model files for the Hangry Rail project.

## Suggested folders

- `hooks/`: General utility hooks and hangers
- `tool-mounts/`: Tool-specific mounts
- `adapters/`: Spacers, clips, and interface parts

## Naming guidance

- Include compatibility or rough size in filenames
- Use versions when you revise geometry (example: `drill-hook-v2.stl`)
- Prefer lowercase and hyphens

## Optional extras

- Add matching `.3mf` files when slicer settings matter
- Add preview images (`.png` or `.jpg`) alongside models
