# STL Library

Use this folder to store model files that back the Hangry Rail project.

## Suggested layout

- `hooks/`: General utility hooks and hangers
- `tool-mounts/`: Tool-specific mounts
- `adapters/`: Spacers, clips, and interface parts

## Naming tips

- Include rough size or compatibility in the filename
- Add version tags (for example: `drill-hook-v2.stl`)
- Keep names lowercase with hyphens for easy linking in docs

## Optional extras

- Include matching `.3mf` files when slicer settings matter
- Add a preview image (`.png` or `.jpg`) for each design
